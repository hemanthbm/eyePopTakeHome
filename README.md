# EyePop Processor

A Python package to process local images using the EyePop.ai API.

## Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

To process a local image:

```bash
python3 main.py ./test.jpg
```

## Running Tests

Run tests with:

```bash
python3 -m unittest discover -s tests
```
